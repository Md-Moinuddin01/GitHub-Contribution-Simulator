GitHub Contribution Simulator

A full-stack practice environment where beginners learn the open-source contribution workflow before contributing to real GitHub repositories.

Open demo workspace Open landing page

How To Use

Run Locally

npm install
npm run db:generate
npm run dev

Then open http://127.0.0.1:3000/auth/demo for instant demo access.

What The App Includes

Repository practice
Fake open-source repositories with issues, README tabs, file structures, and real GitHub reference links.
Issue marketplace
Browse, filter, assign, and work on beginner to advanced practice issues.
Browser code editor
Monaco editor with file tree, syntax highlighting, save/reset, fake tests, and terminal commands.
Git workflow
Practice fork, branch, edit, stage, commit, push, and pull request steps.
AI maintainer review
Get review-style feedback using OpenAI/Gemini when configured, or a local fallback reviewer.
Gamification
XP, levels, badges, contribution heatmap, streak-style activity, and leaderboard screens.

Why This Is Different

Normal GitHub tutorials teach commands one by one. This simulator teaches the whole contribution experience in a safe place: choosing an issue, editing code, committing, opening a PR, receiving review, fixing changes, resolving conflicts, and merging.

  • Beginners can make mistakes without affecting a real project.
  • The UI feels familiar to GitHub users but stays beginner-friendly.
  • AI review gives realistic maintainer feedback before real public review.
  • The editor, issue tracker, PR simulator, terminal, and dashboard live in one app.
  • Teachers and communities can use the admin area as a training console.

Database And AI Notes

The app works immediately in demo mode with ENABLE_DATABASE=false. To use PostgreSQL, set ENABLE_DATABASE=true, configure DATABASE_URL, then run migrations and seed data. 

npm run db:migrate
npm run db:seed

AI review works with OPENAI_API_KEY or GEMINI_API_KEY. If no key is configured, the simulator uses a local maintainer-review fallback.
